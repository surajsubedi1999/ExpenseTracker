# Expense Tracker

A React Native app for managing expenses and tracking financial transactions.

## Features

- Login with admin/admin credentials
- View transaction list
- Add new transactions
- View transaction details
- Logout functionality

## Setup

```bash
# Create project
npx create-expo-app ExpenseTracker --template blank
cd ExpenseTracker

# Install dependencies
npx expo install @react-navigation/native @react-navigation/native-stack react-native-screens react-native-safe-area-context @react-native-picker/picker

# Start development
npm start
